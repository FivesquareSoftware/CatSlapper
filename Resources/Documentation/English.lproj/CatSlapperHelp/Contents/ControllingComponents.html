Controlling Components

CatSlapper allows you to send commands to an instance of Tomcat Manager that instruct it to take various actions on Tomcat components. To issue a command for a component, select the command from the popup menu for a component in the "Commands" column.

To start or stop a Tomcat instance:

Click the popup "Commands" menu for the root component in the "Components" outline view.

Select "Toggle".

To deploy a Web application to a Tomcat instance:

Click the popup "Commands" menu for a host component in the "Components" outline view.

Select "Deploy webapp".

There are various ways to deploy webapps. See Deploying Applications for more information.

To start, stop, reload or undeploy a webapp:

Click the popup "Commands" menu for a Web application component in the "Components" outline view.

Select "Start", "Stop", "Reload" or "Undeploy".

Be careful when undeploying Web applications. This will remove them from the server in most cases. Along with the original WAR file if expansion occurred.
