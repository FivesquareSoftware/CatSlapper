Controlling Tomcat Instances

CatSlapper lets you perform basic management functions on any Tomcat instances in its database.

To stop or start an instance:

Click "Action" from the main menu and select "Toggle", click the "Toggle" toolbar icon, or open the CatSlapper dock menu, click the instance name and select "Toggle" from the sub menu.

To restart an instance:

Click "Action" from the main menu and select "Restart", click the "Restart" toolbar icon, or open the CatSlapper dock menu, click the instance name and select "Restart" from the sub menu.
