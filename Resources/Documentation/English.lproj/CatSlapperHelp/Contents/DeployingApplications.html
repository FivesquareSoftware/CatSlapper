Deploying Applications

The Tomcat Manager application is capable of deploying Web applications to a particular host configured on a Tomcat instance. It can do this by either deploying an application residing on the same machine as itself, or by receiving a WAR file in an upload along with the command to deploy.

To start the deployment process, select "Deploy webapp" from the commands popop menu for a host component.

To deploy a webapp located on the server:

Optionally enter a value for context path.

Optionally enter a URL of the form file://path/to/file for an XML configuration file for this application.

Enter a file URL for the application directory or WAR file you wish to deploy.

Click "Deploy".

To deploy a webapp by uploading a WAR file:

Click "Select" and browse to the WAR file you wish to upload.

Click "Deploy" (This may take a while).
