Getting Started

If you don't have any Tomcats installed (or running) when you first launch CatSlapper you'll be greeted by the main controller window displaying a blank configuration. You can get going several ways.

Install an instance of Tomcat

Click "Action" from the main menu and select "Install", or click the "Install" toolbar icon

This will place Tomcat files on your system and insert the configuration into CatSlapper's management database. See Installing A Tomcat Instance for more information.

Manually enter configuration information

Click "Action" from the main menu and select "Add", or click the "Add" toolbar icon.

Enter the name you wish to give this configuration.

Accept the System default for JAVA_HOME or enter the path to a valid JDK.

Enter a value for CATALINA_HOME that points to a valid Tomcat installation.

Enter optional values as you wish.
