Managing Tomcat Components

Tomcat ships with some default Web applications, among these the "Manager" application. Tomcat Manager allows administrators to perform management tasks on Tomcat components and gather information about deployed Web applications, the system and Tomcat itself using a Web browser.

In order to simplfy the use of the Tomcat Manager application and consolidate its core capabilities into a single interface, CatSlapper utilizes a number of Tomcat Manager's programmatic interfaces, including its JMX (Java Management Extensions) proxy servlet, which uses JMX beans to gather information about and control various components within Tomcat. CatSlapper displays this component information and provides controls for common tasks you might wish to perform in its "Components" view.
