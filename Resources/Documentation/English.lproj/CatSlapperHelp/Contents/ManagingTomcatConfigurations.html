Managing Tomcat Configurations

CatSlapper can keep any number of Tomcat configurations in its management database.

Adding or Removing a Configuration

To add a configuration, click "Action" from the main menu and select "Add", or click the "Add" toolbar icon.

To remove a configuration, click "Action" from the main menu and select "Remove", or click the "Remove" toolbar icon.

Navigating Configurations

To view any configuration, click the dropdown menu in the main controller window and select the name of the configuration you want to see.

To see the next configuration, click "View" from the main menu and select "Next Kitty", or click the "next" toolbar icon.

To see the previous configuration, click "View" from the main menu and select "Previous Kitty", or click the "previous" toolbar icon.
