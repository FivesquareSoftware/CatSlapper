Using the Default Environment

When using CatSlapper to control a single instance of Tomcat, it can be very handy to use a set of defaults for Tomcat's required environment varibles. Often, these are set in your shell. CatSlapper allows you to explicity set default values in a preference pane. You can also choose to have these defaults loaded from your tcsh or bash environment.

  1. Select CatSlapper > Preferences from the main menu.
  2. Select the "Defaults" pane.
  3. Leave "Load from ${SHELL} environment" checked or uncheck it and enter your desired values in the fields provided.

You can reload default values from your shell any time by clicking the refresh button.
