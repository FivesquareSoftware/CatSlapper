Launching Tomcat Instances at Startup

CatSlapper can install a StartupItem, a script that causes Mac OS X to launch a server or other process when the system starts, for any instance of Tomcat it is managing.

StartupItems are located inside /Library/StartupItems. You should not manually edit the StartupItems that CatSlapper creates, as your changes wil be overwritten. CatSlapper tries to synchronize StartupItems when it launches.

To enable or disable a StartupItem for a Tomcat Instance:

Check or uncheck the "Enable StartupItem" checkbox.
